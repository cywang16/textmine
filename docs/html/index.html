Text Mine:

The Text Mine tool kit is a collection of Perl modules and scripts to perform text mining functions. You can use the modules from the command line or through a web based script. Text Mine is a SourceForge project. The current version of the code is on CVS. The first version of Text Mine was released in Sep, 2004. Download Text Mine . To use Text Mine , you will need Perl, Apache, and MySql . Read more about text mining in the book.
